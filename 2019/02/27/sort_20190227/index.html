常用的一些简单排序算法

发表于 2019-02-27 | 分类于 算法

冒泡排序

思想：从某个方向两两相邻比较元素的值，若为逆序，则交换。每一轮冒泡完可以保证在第i个位置的元素是正确的。

插入排序

思想：依次将待排序的目标插入到已经排好序的序列里，直到所有的序列都排好序

选择排序

思路：从无序区选择一个最大/最小的元素插入到有序区的最后。

希尔排序

思想：让序列基本有序，使用分组插入排序。

归并排序

思想：分治法，将数组拆分为n个组排序，然后归并。
